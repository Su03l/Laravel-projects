# مختصر الروابط (URL Shortener API)

## اسم الفكرة

**خدمة اختصار الروابط وتتبع الزيارات (Like Bit.ly)**

## شرح تفصيلي للفكرة

هذا هو المشروع السادس في تحدي "30 يوم 30 مشروع".

النظام يقدم خدمة تحويل الروابط الطويلة والمعقدة إلى روابط قصيرة سهلة المشاركة (مكونة من 6 خانات عشوائية).

بالإضافة إلى الاختصار، يقوم النظام بتتبع عدد النقرات (Visits) على كل رابط، مما يسمح للمستخدم بمعرفة مدى انتشار رابطه.

## المميزات

1. **توليد أكواد فريدة:** استخدام `Str::random(6)` لإنشاء كود قصير غير مكرر.
2. **إعادة التوجيه (Redirection):** استخدام `redirect()->away()` لتحويل المستخدم للرابط الأصلي عند زيارة الكود القصير.
3. **عداد الزيارات (Analytics):** زيادة حقل `visits` تلقائياً (+1) عند كل عملية تحويل ناجحة.
4. **التحقق:** التأكد من أن الرابط المدخل هو URL صالح وحقيقي.

## جدول الروابط (API Routes)

| الطريقة  | الرابط                    | الوصف                   | البيانات المطلوبة |
| :------- | :------------------------ | :---------------------- | :---------------- |
| **GET**  | `/api/links`              | عرض سجل الروابط         | -                 |
| **POST** | `/api/links`              | إنشاء رابط قصير         | `url` (String)    |
| **GET**  | `/api/links/{code}/stats` | إحصائيات رابط معين      | -                 |
| **GET**  | `/api/go/{code}`          | الرابط السحري (التحويل) | -                 |

## التشغيل

```bash
# تثبيت الاعتمادات
composer install

# إعداد قاعدة البيانات
php artisan migrate

# تشغيل السيرفر
php artisan serve
```

## طريقة الاستخدام

1. أرسل `POST` إلى `/api/links` مع الرابط الطويل.
2. احصل على الرابط القصير من الرد (مثلاً: `/api/go/AbC123`).
3. شارك الرابط، وشاهد عداد الزيارات يرتفع!
