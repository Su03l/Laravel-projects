# تحدي 30 يوم 30 مشروع

مجموعة من واجهات برمجة التطبيقات (APIs) المبنية على Laravel لتطبيقات مختلفة.

## المشاريع

| #   | اسم المشروع | الوصف                                     | المميزات الرئيسية                  |
| --- | ----------- | ----------------------------------------- | ---------------------------------- |

## التقنيات المستخدمة

جميع المشاريع مبنية باستخدام:

- **Laravel** - إطار عمل PHP
- **SQLite** - قاعدة البيانات
- **PHP** - لغة البرمجة من جانب الخادم
- **React** - مكتبة JavaScript للواجهات (في بعض المشاريع)
- **Tailwind CSS** - إطار عمل CSS (في بعض المشاريع)
- **Scramble** - توثيق واختبار API تلقائي (في بعض المشاريع)
