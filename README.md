## التقنيات المستخدمة

جميع المشاريع مبنية باستخدام:

- **Laravel** - إطار عمل PHP
- **SQLite** - قاعدة البيانات
- **PHP** - لغة البرمجة من جانب الخادم
- **Scramble** - توثيق API (في بعض المشاريع)
