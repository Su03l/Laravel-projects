# مجموعة مشاريع Laravel

مجموعة من واجهات برمجة التطبيقات (APIs) المبنية على Laravel لتطبيقات مختلفة.

## المشاريع

|  #  | اسم المشروع                                          | الوصف                                            | المميزات الرئيسية                                           |
| :-: | :--------------------------------------------------- | :----------------------------------------------- | :---------------------------------------------------------- |
|  1  | [PhonebookAPI](./PhonebookAPI)                       | إدارة جهات الاتصال الهاتفية مع البحث             | CRUD، Validation، Search                                    |
|  2  | [HRSystem](./HRSystem)                               | نظام الموارد البشرية لإدارة الموظفين والأقسام    | Department Management، Employee Management                  |
|  3  | [UniversitySystem](./UniversitySystem)               | إدارة الطلاب والمواد الدراسية والتسجيل           | Student-Course Relationships، Enrollment                    |
|  4  | [Mini_Store_API](./Mini_Store_API)                   | متجر إلكتروني بسيط مع إدارة مخزون تلقائية        | Stock Management، Transactions، Order Numbers               |
|  5  | [Car_Rental_API](./Car_Rental_API)                   | نظام تأجير سيارات مع حساب تلقائي للتكاليف        | Status Tracking، Auto Billing، Conflict Prevention          |
|  6  | [Project_Manager](./Project_Manager)                 | إدارة المشاريع والمهام مع ملفات وملاحظات         | Polymorphic Relations، File Uploads                         |
|  7  | [RealEstate-API](./RealEstate-API)                   | بحث متقدم عن العقارات مع فلترة ديناميكية         | Query Scopes، Fuzzy Search، Scramble Docs                   |
|  8  | [pixel-positions](./pixel-positions)                 | منصة وظائف تربط أصحاب العمل بالباحثين            | Authentication، Tags، Tailwind CSS                          |
|  9  | [todo-app](./todo-app)                               | قائمة مهام مع نظام مصادقة متكامل                 | Sanctum، Profile Management، CSRF Protection                |
| 10  | [Secure_Notes_API](./Secure_Notes_API)               | ملاحظات شخصية آمنة مع حماية البيانات             | Token Auth، Scoped Queries، Data Ownership                  |
| 11  | [Task_Manager_with_Roles](./Task_Manager_with_Roles) | نظام مهام مع صلاحيات Admin/User                  | RBAC، Policies، Scoped Access                               |
| 12  | [ECommerce-API](./ECommerce-API)                     | متجر إلكتروني متقدم مع Transactions وإدارة ملفات | DB Transactions، Race Conditions، Smart Stock، File Uploads |
| 13  | [University-System_ad](./University-System_ad)       | نظام تسجيل جامعي ذكي مع إدارة شاملة للطلاب       | Student Management، Polymorphic Relations، Advanced Search  |

## التقنيات المستخدمة

جميع المشاريع مبنية باستخدام:

- **Laravel** - إطار عمل PHP
- **SQLite** - قاعدة البيانات
- **PHP** - لغة البرمجة من جانب الخادم
- **Scramble** - توثيق API (في بعض المشاريع)
