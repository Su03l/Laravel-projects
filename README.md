# مجموعة مشاريع Laravel

مجموعة من واجهات برمجة التطبيقات (APIs) المبنية على Laravel لتطبيقات مختلفة.

## المشاريع

### 1. [PhonebookAPI](./PhonebookAPI)

واجهة برمجة تطبيقات RESTful لإدارة جهات الاتصال الهاتفية مع إمكانية البحث.

**المميزات الرئيسية:**

- عمليات CRUD كاملة لجهات الاتصال
- البحث عن جهات الاتصال برقم الهاتف
- التحقق من صحة البيانات وسلامتها

### 2. [HRSystem](./HRSystem)

واجهة برمجة تطبيقات RESTful لإدارة عمليات الموارد البشرية بما في ذلك الموظفين والأقسام.

**المميزات الرئيسية:**

- إدارة الأقسام
- إدارة الموظفين مع ربطهم بالأقسام
- معالجة شاملة لبيانات الموارد البشرية

### 3. [UniversitySystem](./UniversitySystem)

واجهة برمجة تطبيقات RESTful لإدارة عمليات الجامعة بما في ذلك الطلاب والمواد الدراسية.

**المميزات الرئيسية:**

- إدارة الطلاب والمواد الدراسية
- تسجيل المواد والالتحاق بها
- معالجة العلاقة بين الطلاب والمواد

### 4. [Mini_Store_API](./Mini_Store_API)

واجهة برمجة تطبيقات RESTful لنظام تجارة إلكترونية بسيط يدير المنتجات والطلبات مع إدارة تلقائية للمخزون.

**المميزات الرئيسية:**

- إدارة المنتجات مع تتبع المخزون
- معالجة الطلبات مع دعم منتجات متعددة
- إدارة تلقائية للمخزون (خصم/استعادة الكميات)
- معاملات قاعدة البيانات لضمان سلامة البيانات
- توليد أرقام فريدة للطلبات

### 5. [Car_Rental_API](./Car_Rental_API)

واجهة برمجة تطبيقات RESTful قوية لإدارة أعمال تأجير السيارات مع حساب تلقائي للتكاليف وإدارة صارمة لحالة المركبات.

**المميزات الرئيسية:**

- إدارة الأسطول مع تتبع الحالة (متاح، مؤجر، صيانة)
- إدارة العملاء مع تحديد فريد للهوية
- حساب تلقائي للفواتير بناءً على مدة الإيجار
- إدارة الحالة (تغيير حالة السيارة)
- منع التعارضات للمركبات المؤجرة
- معاملات قاعدة البيانات لضمان سلامة البيانات

### 6. [Project_Manager](./Project_Manager)

نظام مرن لإدارة المشاريع مبني باستخدام Laravel. يستعرض مفاهيم متقدمة مثل العلاقات متعددة الأشكال (Polymorphic Relationships) لربط الملاحظات والملفات بالمشاريع والمهام ديناميكياً.

**المميزات الرئيسية:**

- إدارة المشاريع والمهام مع تتبع حالة الإنجاز
- ملاحظات ذكية (Polymorphic) للمشاريع والمهام
- رفع وتخزين الملفات مع روابط كاملة للاستخدام
- حذف نظيف للملفات من السيرفر عند الحذف
- نقاط وصول موحدة للمشاريع والمهام

### 7. [RealEstate-API](./RealEstate-API)

نظام API مرن لإدارة العقارات يركز على نطاقات الاستعلام (Query Scopes) والفلترة الديناميكية. يتيح البحث عن العقارات بناءً على معايير متعددة ومعقدة بكفاءة عالية.

**المميزات الرئيسية:**

- محرك بحث متقدم مع فلترة حسب المدينة، النوع، الحالة، السعر والمساحة
- سكوبات ذكية (Smart Scopes) باستخدام Local Scopes
- بحث تقريبي (Fuzzy Search) للمساحة بنطاق تسامح (±20 متر)
- استخدام FormRequest للتحقق من صحة البيانات
- توثيق تلقائي للـ API باستخدام Scramble
- حل تعارض الروابط بين الروابط المخصصة وروابط الموارد القياسية

### 8. [pixel-positions](./pixel-positions)

منصة شاملة للوظائف تربط أصحاب العمل بالباحثين عن العمل من خلال واجهة سهلة الاستخدام.

**المميزات الرئيسية:**

- نظام نشر وإدارة الوظائف
- مصادقة وتفويض المستخدمين
- تصنيف الوظائف بواسطة الوسوم
- وظيفة بحث متقدمة
- ملفات تعريف أصحاب العمل مع قوائم الوظائف
- واجهة مستخدم حديثة باستخدام Tailwind CSS

### 9. [todo-app](./todo-app)

واجهة برمجة تطبيقات RESTful لإدارة قوائم المهام مع نظام مصادقة متكامل باستخدام Laravel Sanctum.

**المميزات الرئيسية:**

- مصادقة المستخدمين باستخدام Laravel Sanctum
- إدارة كاملة للمهام (CRUD)
- إدارة الملف الشخصي للمستخدم
- تغيير كلمة المرور
- حماية CSRF للتطبيقات أحادية الصفحة (SPA)
- دعم المصادقة المستندة إلى الكوكيز

### 10. [Secure_Notes_API](./Secure_Notes_API)

نظام واجهة برمجية (API) آمن ومتكامل لإدارة الملاحظات الشخصية. يعتمد المشروع على Laravel Sanctum للمصادقة وحماية البيانات، مما يضمن أن كل مستخدم يمكنه الوصول إلى بياناته الخاصة فقط.

**المميزات الرئيسية:**

- نظام مصادقة متكامل باستخدام Tokens
- حماية البيانات (Data Ownership) مع Scoped Queries
- بحث آمن يمنع تداخل البيانات بين المستخدمين
- التحقق من الصلاحيات لحماية بيانات المستخدم
- حماية كاملة لنقاط الوصول باستخدام Middleware

## التقنيات المستخدمة

جميع المشاريع مبنية باستخدام:

- **Laravel** - إطار عمل PHP
- **SQLite** - قاعدة البيانات
- **PHP** - لغة البرمجة من جانب الخادم
- **Scramble** - توثيق API (في بعض المشاريع)
